# Time Series Forecasting (Machine Learning Project)

## Overview

This project analyzes historical time series data to evaluate forecasting techniques using Python. It focuses on predicting trends in real-world datasets such as daily temperature and sales.

---

## Problem

Forecasting future values in time series data is essential for understanding trends and making data-driven decisions.

---

## Approach

The project implements two forecasting methods:

* **Naive Forecasting** (previous value prediction)
* **Simple Moving Average (SMA)** with a 7-day window

Data preprocessing was performed using Pandas, and model performance was evaluated using Mean Absolute Error (MAE).

---

## Results

* Naive Forecast captures short-term patterns
* SMA smooths the data and reveals trends
* Forecast performance evaluated using MAE
